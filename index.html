Proyek Saya

Website portofolio responsif

Sebuah website portofolio yang responsif yang dibuat untuk seorang desainer UI/UX. Proyek ini mencakup perancangan tata letak visual (desain grafis) menggunakan figma, diikuti dengan pengembangan front-end menggunakan HTML, CSS, dan JavaScript.

Aplikasi manajemen proyek

Pengembangan sebuah aplikasi manajemen proyek berbasis web. Dari sisi desain grafis, saya bertanggung jawab membuat antarmuka yang intuitif dan mudah digunakan, termasuk ikon kustom, skema warna, dan panduan gaya (style guide).
